# Contributing

Thanks for your interest in improving this skill!

## How to Contribute

### Reporting Problems

Found a bug or have a suggestion? [Open an issue](https://github.com/20kiki/mineru-pdf/issues) and choose the appropriate template.

### Improving the Skill

1. Fork the repo and create a feature branch
2. Make your changes to `SKILL.md`
3. Test the skill in Claude Code with at least one complex PDF
4. Open a pull request with a summary of what you changed and why

### Skill Guidelines

- The skill should remain focused on **content extraction only**. Editing operations belong in the official pdf skill
- Keep `SKILL.md` under 150 lines — be concise
- Add new capabilities as separate sections, not scattered inline
- If adding scripts, place them in a `scripts/` directory and reference them from `SKILL.md`

### Commit Style

Use conventional commits: `feat:`, `fix:`, `docs:`, `refactor:`, `chore:`
